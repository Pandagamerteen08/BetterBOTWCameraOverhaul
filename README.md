# Overview

**Better Breath of the Wild (BOTW) Camera Overhaul** is a complete enhancement and expansion of *The Legend of Zelda: Breath of the Wild's* camera system.

The goal of this project is not to replace the identity of Breath of the Wild's camera. Nintendo created one of the most carefully designed third-person cameras in modern gaming, balancing exploration, combat, traversal, and cinematic presentation.

Instead, this project improves and expands that foundation.

Better BOTW Camera Overhaul provides players with complete control over nearly every aspect of the camera while preserving the original Breath of the Wild feeling.

From simple quality-of-life improvements such as disabling camera shake and adjusting distance, to advanced features such as first-person mode, free camera controls, cinematic presets, and full camera customization, this mod transforms the camera into a fully configurable system.

# Project Ecosystem and Distribution Structure

Better Breath of the Wild (BOTW) Camera Overhaul is designed as more than a traditional game modification.

The project is separated into multiple components to provide the best experience for different types of users.

The main goal is to keep the Nintendo Switch mod lightweight, fast, and easy to install while providing advanced creator tools separately for users who want to create cinematic content.

The project consists of three primary components:

1. Better Breath of the Wild (BOTW) Camera Overhaul
2. Better BOTW Camera Studio
3. Better BOTW Cinematic Content Packs

# Why The Switch Mod Remains Small

Even though the project contains hundreds of customization options, the mod remains relatively small because most features are controlled through:

- Code
- Configuration values
- Mathematical calculations
- Runtime adjustments

Camera systems do not require large textures, models, or audio files.

The majority of the project is logic rather than traditional game assets.
